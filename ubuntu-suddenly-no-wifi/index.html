关于 Ubuntu 突然无法连接 Wifi 的解决方案

2014-09-27 | Memo

事实上我也不知道发生了什么，大概是几天前插了“小度 Wifi”的缘故。没有任何征兆地，Wifi 就用不了了。其实我也不知道原理，大概是某个驱动被刷掉了。下面是从网上找来的答案：

sudo apt-get install wicd-daemon

做个记录。
